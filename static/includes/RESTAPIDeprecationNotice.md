{{< hint type="info" title="REST API Deprecation Notice" >}}
The TrueNAS REST API was deprecated in TrueNAS 25.04.
Full removal of the REST API is planned for TrueNAS 26.

Starting with TrueNAS 25.10.1, the system displays alerts when deprecated REST API endpoints are accessed to help identify integrations that require migration to the JSON-RPC 2.0 over WebSocket API.
{{< /hint >}}
