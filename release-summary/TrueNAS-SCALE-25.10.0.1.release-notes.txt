## TrueNAS 25.10.0.1

This maintenance release resolves critical SMB file sharing issues, application management bugs, and improves overall system stability.

- **Critical Bug Fixes:** Fixes include SMB2 directory enumeration causing files to appear missing or inaccessible, Time Machine snapshot failures, Docker startup timeouts on HDD systems, and custom app update failures.

**Pre-Update Actions:** None required.
