## TrueNAS 25.10.2.2

This maintenance release resolves pool creation failures on systems with a USB boot drive, log VDEV removal errors, bond status alert check failures, and an NTB memory handling issue affecting High Availability configurations.

- **Critical Bug Fixes:** Fixes an error that prevented new storage pool creation on systems where the TrueNAS boot pool resides on a USB device. Resolves log VDEV removal failures, bond status alert check errors caused by a busy Netlink socket, and an NTB window memory handling issue during link state changes on HA systems.

**Pre-Update Actions:** None required.
