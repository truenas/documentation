TrueNAS includes an application catalog for deploying and managing containerized applications.

{{< include file="/static/includes/apps/AppsMarket.md" >}}

The [TrueNAS Applications Market](https://apps.truenas.com/) is the primary resource for app management tutorials and individual app deployment guides.

- For initial setup, storage configuration, app security, and support level definitions, visit [Getting Started with Apps](https://apps.truenas.com/getting-started/).
- For day-to-day tasks such as discovering, installing, and managing apps, visit [Managing Apps](https://apps.truenas.com/managing-apps/).
- To search available applications and find app-specific tutorials and reference content, browse the [Catalog](https://apps.truenas.com/catalog/).

This section contains UI reference documentation for the TrueNAS **Apps** screens and settings.
